### how to init

* [prepare session.txt](https://github.com/wimglenn/advent-of-code-wim/issues/1)
* Build initaoc

    ```shell
    cd initaoc
    go build -o initApp
    mv initApp ../
    cd ..
    ./initApp -y {year} -d {day}
    ```
